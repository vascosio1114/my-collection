# 🏆 My Collection Vault

> A personal web app to catalogue and show off my collectibles — NBA cards, Pokémon cards, jerseys, toys, and more.

🔗 **[Live Site](https://vascosio1114.github.io/my-collection/)**

Built this mainly because I wanted a proper place to track my collection, and it turned out to be a good way to practice building a full frontend app with vanilla JS.

---

## ✨ Features

- **📦 Dashboard** — total items, estimated portfolio value, top item, active categories at a glance
- **🔍 Search & Filter** — search by name, notes, or year; filter by category (NBA Card, Pokémon Card, Jersey, Toy, Other)
- **🔀 Sort** — by newest, oldest, highest or lowest value
- **📊 Data Insights** — donut chart breakdown of collection value by category
- **🖼️ Grid / List View** — toggle between card grid and list layout
- **🔐 Admin Panel** — password-protected panel to add, edit, and manage items
- **📱 Responsive** — works on desktop and mobile

---

## 🛠️ Built With

HTML · CSS · JavaScript (Vanilla) · GitHub Pages

## 🚀 Run Locally

```bash
git clone https://github.com/vascosio1114/my-collection.git
cd my-collection
open index.html
```

No build step needed — fully static.
